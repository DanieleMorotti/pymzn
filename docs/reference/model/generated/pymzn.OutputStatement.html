pymzn.OutputStatement¶

class pymzn.OutputStatement(output, comment=None)¶

An output statement.

output¶

str – The content of the output statement, i.e. only the actual output without the starting ‘output’, the square brackets and the ending semicolon.

comment¶

str – A comment to attach to the output statement.
