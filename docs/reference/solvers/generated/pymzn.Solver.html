pymzn.Solver¶

class pymzn.Solver(support_ozn, support_all, globals_dir=None)¶

Abstract solver class.

All solver classes inherit from this class and provide implementations for the solve method.

support_ozn¶

bool – Whether the solver’s output can be parsed by solns2out.

support_all¶

bool – Whether the solver supports the output of all solutions.

globals_dir¶

str – The directory containing the solver-specific redefinitions of global constraints, used when calling mzn2fzn. This should be the default for the solver. When None is provided (default) the std directory is used.
