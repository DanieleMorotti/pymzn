pymzn.process.Process.run¶

Process.run(input=None, timeout=None)¶

Run the process synchronously.

Parameters:
  • input (str or bytes or None) – The content to for the input stream of the process.
  • timeout (float or None) – The timeout for the process. None means no timeout.
